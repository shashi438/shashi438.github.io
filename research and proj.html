Research

MS Thesis - Self supervised Representation Learning using Frame Order Prediction

The success of deep learning models in challenging tasks of computer vision and NLP depend on good vector representations of data. We introduced a novel self-supervised task which help our network to learn robust weights for action recognition on videos. Our neural network has two tasks; predicting if a tuple of input frames is temporally consistent, and if not, predict the positioning of incorrect tuple.
The efficacy is finally measured by using these trained networks on downstream tasks which is action recognition on standard datasets UCF101 and HMDB51.
Our model improves on the baseline accuracy by 5%, while using a fraction of the data required.
